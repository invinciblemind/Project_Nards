# Nardy

**Nardy** — приложение для игры в нарды на Python 3.12.

В проекте реализованы длинные и короткие нарды, графический интерфейс, локальная игра на одном компьютере и LAN-режим для игры в двух отдельных окнах или на двух компьютерах в одной сети.

## Возможности

* выбор режима игры:
  * длинные нарды;
  * короткие нарды;
* графический интерфейс на `tkinter`;
* отображение игрового поля, шашек, бара, снятых шашек и кубиков;
* бросок кубиков;
* подсветка доступных ходов;
* выполнение ходов мышью;
* обработка дублей;
* взятие шашек в коротких нардах;
* снятие шашек с доски;
* откат последнего завершённого хода;
* экран победы;
* русский и английский интерфейс;
* LAN-режим через TCP-сокеты.

## Установка

Требуется Python 3.12 или новее.

Скачайте проект и перейдите в его папку:

```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd Project_Nards
```

Создайте виртуальное окружение.

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Установите приложение:

```bash
python -m pip install --upgrade pip
python -m pip install .
```

На Linux для работы графического интерфейса может потребоваться пакет `tkinter`:

```bash
sudo apt install python3-tk
```

## Запуск

Запустить приложение:

```bash
nardy
```

Запустить с русским интерфейсом:

```bash
nardy --locale ru
```

Запустить с английским интерфейсом:

```bash
nardy --locale en
```

Также можно запустить приложение как Python-модуль:

```bash
python -m nardy
```

## Сборка из чистого клона

Полная сборка проекта выполняется автоматически через `doit`.

1. Установите зависимости разработки:

   ```bash
   python -m pip install -e .[dev]
   ```

2. Запустите полный набор проверок:

   ```bash
   python -m doit check
   ```

   Эта команда запускает:

   * проверку стиля `flake8`;
   * проверку docstring через `pydocstyle`;
   * тесты;
   * отчёт покрытия;
   * сборку Sphinx-документации;
   * извлечение, обновление и компиляцию переводов;
   * сборку wheel-дистрибутива.

3. Соберите только wheel-дистрибутив, если проверки уже не нужны:

   ```bash
   python -m doit build
   ```

Готовый wheel появится в папке `dist/`. Скомпилированные переводы `.mo`
и шаблон `messages.pot` не хранятся в git: они генерируются командами
`python -m doit babel_compile` и `python -m doit build`.

Установить собранный wheel можно так:

```bash
python -m pip install dist/nardy-0.1.0-py3-none-any.whl
```

## Локальная игра

Для локальной игры запустите:

```bash
nardy --locale ru
```

В главном меню выберите режим:

* длинные нарды;
* короткие нарды.

После выбора режима откроется игровое поле. Игрок бросает кубики и выбирает доступные ходы мышью.

Короткий сценарий партии:

1. Выберите режим `Long Nardy` или `Short Nardy`.
2. Нажмите кнопку броска кубиков.
3. Нажмите на шашку, которую хотите передвинуть.
4. Нажмите на подсвеченную доступную позицию.
5. Повторите ходы, пока доступны значения кубиков.
6. После завершения хода очередь перейдёт второму игроку.
7. Побеждает игрок, который первым снимет все свои шашки с доски.

## LAN-игра

LAN-режим позволяет играть в двух отдельных окнах или на двух компьютерах в одной локальной сети.

### Запуск на одном компьютере

В первом терминале запустите сервер:

```bash
nardy --server --socket-host 127.0.0.1 --socket-port 8765 --locale ru
```

Во втором терминале подключитесь к серверу:

```bash
nardy --join --socket-host 127.0.0.1 --socket-port 8765 --locale ru
```

Игрок, запустивший сервер, играет белыми шашками. Второй игрок подключается чёрными шашками.

После подключения оба окна работают как обычная партия: каждый игрок ходит
только в свою очередь, а состояние доски синхронизируется через TCP-сокет.

### Запуск по локальной сети

На компьютере-хосте:

```bash
nardy --server --socket-host 0.0.0.0 --socket-port 8765 --locale ru
```

На втором компьютере:

```bash
nardy --join --socket-host IP_АДРЕС_ХОСТА --socket-port 8765 --locale ru
```

Например:

```bash
nardy --join --socket-host 192.168.1.10 --socket-port 8765 --locale ru
```

## Управление

Основное управление выполняется мышью:

* нажмите кнопку броска кубиков;
* выберите шашку;
* выберите подсвеченный доступный пункт для хода;
* после завершения доступных ходов очередь переходит к следующему игроку.

Доступен откат последнего завершённого хода через кнопку undo.

## Языки интерфейса

Поддерживаются две локали:

* `ru` — русский язык;
* `en` — английский язык.

Пример запуска на русском языке:

```bash
nardy --locale ru
```

## Возможные проблемы

### Команда `nardy` не найдена

Убедитесь, что приложение установлено:

```bash
python -m pip install .
```

И что активировано виртуальное окружение.

### Не запускается графический интерфейс на Linux

Установите `tkinter`:

```bash
sudo apt install python3-tk
```

### Второй игрок не подключается в LAN-режиме

Проверьте:

* правильно ли указан IP-адрес хоста;
* открыт ли порт `8765`;
* находятся ли компьютеры в одной сети;
* не блокирует ли подключение firewall.
